# 복잡도

### 시간 복잡도 - 알고리즘에 대해 얼마나 오래 걸리는지
빅오 표기법(Big O)
O(1)      - 상수 시간
O(lonN)   - 로그시간      - 이진 탐색(탐색 범위가 100억 1000억 넘어가면)
O(N)      - 선형시간
O(NlogN)  - 로그 선형시간
O(N^2)    - 이차 시간


### 공간 복잡도 - 특정한 크기의 입력에 대해 알고리즘이 얼마나 많은 메모리를 차지하는지
보통 128 ~ 512MB
보통 리스트 많이 사용 - 만약 리스트 크기가 1000만 단위 이상일 경우 - 다른 방법 고민
